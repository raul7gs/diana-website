DATA INTERACTIVE ANALYSIS AND NAVIGATION APPLICATION

DIANA is desgined to allow users to interactively analyze and visualize data. It helps to compare various design points, enhancing understanding of results.

ENABLING INNOVATIVE CONCEPTS

CPACS can be used to model all kinds of aircraft configurations ranging from conventional wing and tube design to highly unconventional configurations such as the Blended Wing Body.

COLLABORATIVE MDO

CPACS is used for the exchange of data in multi-disciplinary design optimization processes. In combination with an integration framework such as RCE, a complex aircraft design process can be setup containing expert knowledge from different fields/disciplines.
